## Supported Versions

Pictogram currently supports Minecraft versions **1.21.9 through 1.21.11**.

## MOTD Feature

The MOTD feature relies on behaviour that was patched in a later snapshot. As of [26.1 Pre-Release 1](https://www.minecraft.net/en-us/article/minecraft-26-1-pre-release-1), image MOTDs no longer work and this cannot be worked around. Pictogram will continue to be updated and maintained for all other features going forward.

<Warning>
  Image MOTDs will not work on any version above 1.21.11.
</Warning>
